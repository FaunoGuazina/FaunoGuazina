  • ICAM - Actual

  • RDGORS - En Producción
    Microservicio REST para gestión del Registro de Centros Sanitarios (sustituye a RCS) Desarrollo de un microservicio REST especializado en la gestión del registro de centros sanitarios autorizados en Cataluña, en línea con los requerimientos del Registro General (RCS/REGCESS) del Ministerio de Sanidad. Testing: cobertura garantizada con JUnit. RDGORS permite agregar centros autorizados y mantener datos actualizados, alineado con los procesos requeridos por REGCESS. Resultado Microservicio autónomo y reutilizable. Mejora trazabilidad y transparencia del registro. Automatiza parte del cumplimiento normativo. Facilita interoperabilidad entre sistemas sanitarios autonómicos y la base de datos nacional

2024-2024 | Proyecto Final - WhataCook
Backend Developer
WhatACook es una aplicación de recetas desarrollada con Java y Spring Boot, utilizando una arquitectura de microservicios. El backend gestiona la creación, consulta y modificación de recetas, categorías e ingredientes, permitiendo búsquedas avanzadas por nombre e ingredientes. Implementa Spring WebFlux para manejar operaciones asíncronas y reactivas, y Spring Security con JWT para la autenticación y autorización de usuarios, protegiendo los endpoints y datos sensibles. La aplicación utiliza MongoDB como base de datos NoSQL, con Hibernate y JPA para la persistencia de datos. Además, incluye paginación personalizada y búsquedas avanzadas con filtros configurables. La calidad del código se asegura mediante JUnit5 y Mockito, logrando una alta cobertura en pruebas unitarias y mock tests. El proyecto es altamente escalable y está diseñado con API RESTful eficientes, lo que permite la integración con otros sistemas y la gestión avanzada de recetas.
